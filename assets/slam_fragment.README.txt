PLACEHOLDER — spoken-word audio for the slam fragment.

Drop your recording here and name it exactly:

    slam_fragment.mp3

That's it. When someone clicks "▶ perform" in the voice section, this
audio plays alongside the on-screen performance. It stops when they
close performance mode (✕ or Esc).

Prefer a different name or format (.wav, .m4a, .ogg)? Put the file in
this assets/ folder and point "slam_audio" in content/config.json at it,
e.g.  "slam_audio": "assets/my-recording.wav"

Until the file exists, perform mode just runs silently — nothing breaks.
